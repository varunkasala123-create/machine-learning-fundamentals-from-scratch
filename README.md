# machine-learning-fundamentals-from-scratch
🧠 Repository Description (README Intro)

A foundational machine learning project demonstrating core concepts behind regression models through both manual implementation and library-based approaches.

The project includes a from-scratch implementation of linear regression using gradient descent, alongside a more advanced pipeline using polynomial regression and regularization techniques to illustrate overfitting and model generalization.
